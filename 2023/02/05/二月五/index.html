二月五

发表于 2023-02-05 分类于 天亮以前 阅读次数：

你是黑夜里的梦想家，闭上双眼的时候，你是你自己世界里的王者。
你是白天里的吸血鬼，阳光打在你身每一处，都会让你皮开肉绽。

当你还在肆意挥霍着冬日的旧时光时，春天已经悄然而至。
你从没有如此刻一样，希望太阳慢点升起。
你也从来没有如此刻一样，希望冬天慢点结束。

你找了一万个借口，想要留在这个温室。
你说外面太冷了，阳光太刺眼了。
你喜欢在黑夜里被温暖拥抱，以及桌前的台灯，耳机里的音乐。

你其实是渴望永恒的吧？
你希望雪地的篝火可以永远燃烧，给你带来最朴素的安全感。
你希望耳机里的音乐永远悦耳，给你带来无尽的新鲜感。
但你也是心知肚明吧。
It won’t last long.

你清醒的觉察到自己已经被麻痹了很久，
黑洞有引力，需要逃逸速度。
你害怕这就是你的惯性人生。

你，是你吗？
你，是我吧。

二月一
